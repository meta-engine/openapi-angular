# Contributing

This repository is for documentation and issue tracking.

## Reporting Issues

Please report bugs and feature requests via [GitHub Issues](https://github.com/meta-engine/openapi-angular/issues).

When reporting a bug, please include:
- Package version (`@metaengine/openapi-angular` version)
- Node.js version (`node --version`)
- .NET runtime version (`dotnet --version`)
- Angular version
- OpenAPI specification (minimal reproducible example)
- Command or code used
- Expected behavior vs actual behavior
- Error messages and stack traces

## Feature Requests

We welcome feature requests! Please search existing issues first to avoid duplicates.

When suggesting a feature:
- Describe the use case and problem it solves
- Provide examples of how the API would look
- Explain why this would benefit other users

## Questions and Support

For general questions and support:
- Check the [documentation](https://www.metaengine.eu/packages/openapi-angular)
- Search [existing issues](https://github.com/meta-engine/openapi-angular/issues)
